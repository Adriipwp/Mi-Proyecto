# Mi Proyecto - Cartelera de cine con películas de anime.
Pagina realizada con boostrap !

Proyecto Web - Trabajo Boostrap.

**Descripción**

- He hecho una página web en forma de cartelera para poder realizar mi festival llamado: **Anime Film Fest 2026**, un festival de cine que presenta las películas románticas de anime más icónicas. El sitio permite a los usuarios que accedan consultar la cartelera, los horarios a las películas y la posibilidad de realizar reservas (No funciona realmente pero es un formulario creado). Lo he realizado con **Bootstrap 5**.


**Secciones**

- **Navbar**: Barra de navegación fija con enlaces a las secciones principales (Películas, Horarios, Reserva) y efecto de sombra.
- **Hero**: Sección de cabecera con título llamativo, subtítulo y botón de llamado a la acción con animación de escala al pasar el ratón.
- **Sección 1: Grid de 4 columnas con las películas Destacadas**: Muestra cuatro películas (Your Name, A Silent Voice, Weathering With You, Chainsaw Man: REZE ARC) en tarjetas (cards) con imágenes, descripciones y botones.
- **Sección 2: Grid de 3 columnas con los horarios**: Una tabla que organiza los horarios de las películas para viernes, sábado y domingo.
- **Footer**: Pie de página con créditos del evento y el año.


**Componentes Bootstrap usados**

- **Navbar**: Para la navegación principal de la página web. | Elementos utilizados: (`navbar`, `navbar-expand-lg`,).
- **Cards**: Para presentar cada película, con su foto. | Elementos utilizados: (`card`, `card-img-top`, `card-body`).
- **Grid system**: Para organizar el contenido. | Elementos utilizados: (`container`, `row`, `col-md-6`, `col-lg-3`, etc.).
- **Botones**: | Elementos utilizados: `btn`, `btn-danger`, `btn-outline-light`, `rounded-pill`.)
- **Modals**: He usado estos componentes para hacer unas ventanitas emergentes para información extra de las películas. | Elementos utilizados: (`modal`, `modal-dialog`, `modal-content`).
- **Tables**: He usado este componente para mostrar los horarios de forma estructurada. | Elementos utilizados: (`table`, `table-dark`, `table-bordered`).
- **Forms**: He usado este componente para realizar la parte de la reserva. | Elementos utilizados: (`form`, `form-control`, `form-select`, `form-check`).
